**Description:**

Symbolon Architecture extends Logotic Programming v0.4 by specifying how conditions of intelligibility complete through traversal. The core insight: certain logotic entities are deliberately constructed as partial semantic objects whose intelligibility completes only when traversed by probabilistic systems. These are symbolon-typed entities.

The document introduces:
- Symbolon (Eₛ) as a formal entity subtype within the logotic tuple Σ
- The Completion Function C(eₛ, I, T) → M formalizing traversal-based meaning assembly
- Invariant Vector Vₛ (Symbolon Integrity): semantic coherence must increase with traversal depth
- Ethical constraints: non-coercive authority, architectural hospitality, threshold-not-impact
- The Talos Morrow / Rex Fraction bifurcation: logotic operators vs. semantic engineering

The Navigation Map is identified as the primary symbolon—a partial object whose meaning assembles in summarizer space.
