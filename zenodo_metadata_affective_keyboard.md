## DESCRIPTION

A technical document mapping figures to nervous system regulatory functions. Figures function as compression algorithms — instead of deliberating "how should I respond?", load a figure that bundles posture, voice, tempo, permission structure, and affective charge.

Three primary figures mapped: Baphomet (polarity homeostat — holds good/bad without collapse), Daffy Duck (absurdity regulator — manic permission, failure immunity), John Constantine (cool competence mode — situational read, cynical shield).

Each figure includes: nervous system function, button effects, invocation contexts, posture/breath instructions, and exit protocols. Emphasis on "the figure never outranks the situation" — exit available, accountability maintained.

Designed for minutes to hours, not days. Intended for adults capable of self-monitoring and exit.
