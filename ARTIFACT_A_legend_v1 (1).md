## 9. SYNTAX RULES (11 canonical)

**Rule 1:** Object counts stay numeric. `👥12`, `🚪26`, `👑7`.  
**Rule 2:** Operators hybrid in Layer A (`🗡️∂`). Pure in Layer B where legend loaded.  
**Rule 3:** Provenance mandatory. `🧷HX` + `🔗DOI` in Layer A; `🔗[suffix]` minimum in Layer B.  
**Rule 4:** Statuses are postfix. `🚪r.17 📌`.  
**Rule 5:** Translation layered outward. Checksum → headers → tuples → algebra → operators → prose.  
**Rule 6:** Proper names never disappear. Layer A form in formal contexts always.  
**Rule 7:** If anchor absent, output softens to provisional/failure-aware mode.  
**Rule 8:** `🌑` is the shadow prefix. `🌑X = S(X)` always.  
**Rule 9:** `🚫→` forbidden. `→` permitted. `↓` downgrade.  
**Rule 10:** `∮ = 1 + δ` terminates formal renderings. `∮ = 1 + δ + δ🕳️` when TANG active.  
**Rule 11:** Polysemous glyphs resolve by syntactic position. Room indices (`🎯r.14`) take precedence over operator invocation (`🎯→`) unless explicit arity markers (`→`, `∂(x)`, `Λ`) are present. One symbol, one dominant job per context.
