## How to use this bundle

1. **As a research archive:** the bundle preserves the complete state of the Restored Academy Protocol Registry at v1.0 launch (2026-05-21). Future researchers can cite this DOI for the entire registry as it existed at this moment.

2. **As a deployment artifact:** the site files can be deployed to any static host (Vercel, Netlify, GitHub Pages). The `vercel.json` is pre-configured.

3. **As a curriculum resource:** the protocol cards in `protocols/` use a consistent 11-section schema (see `registry-schema-v0.1.md`) and can be used directly as instructional materials.

4. **As a reference catalog:** the cards cite source DOIs for every protocol. All sources are accessible at zenodo.org under the `crimsonhexagonal` community.
