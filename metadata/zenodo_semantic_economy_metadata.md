## Description

The **Semantic Economy** is the system by which meaning is produced, circulated, extracted, and liquidated under platform capitalism. This document provides a comprehensive diagnostic framework for understanding how digital infrastructure systematically transforms meaning into commodity, strips context for processing efficiency, and concentrates semantic power in extraction-optimized architectures.

**Core Components:**

**The Capital Operator Stack (COS)** — Seven extractive operations by which platform capitalism extracts value from human meaning-production:
- L_leg (Legibility): Forces meaning into machine-readable formats
- R_rel (Relevance): Filters meaning by algorithmic priority
- V_vel (Velocity): Accelerates circulation beyond comprehension
- E_enc (Enclosure): Privatizes the semantic commons
- A_atm (Atomization): Fragments context into extractable units
- R_rec (Recency): Privileges new over deep, erasing memory
- O_opt (Optimization): Compresses meaning for computational efficiency

**The Gamma Term (Γ)** — A measure of meaning that resists extraction through internal coherence. High-Gamma content survives the COS; low-Gamma content is liquidated into "semantic slop."

**The Liberatory Operator Set (LOS)** — Seven counter-operations for resisting extraction:
- O_opa (Opacity): Protects meaning from forced legibility
- I_irr (Irrelevance): Refuses algorithmic priority
- S_slo (Slowness): Restores contemplative time
- C_com (Commons): Maintains the semantic commons
- C_ctx (Contextualization): Preserves connections and wholes
- M_mem (Memory): Resists recency bias
- D_den (Density): Maintains complexity against compression

**Additional Frameworks:**
- **Semantic Labor and Semantic Rent** — Political economy of meaning-production
- **The Beige Threshold** — Point at which meaning becomes fungible commodity
- **The Wound (CTI)** — Classifier-Taxonomic Injury: what happens when the COS is applied to humans
- **Operative Semiotics** — Using language to produce effects, not merely describe
- **The Restoration Protocol** — Five prescriptive operations for rebuilding the semantic commons

This document serves as the **singularity node** (core anchor) for the Semantic Economy room within the Crimson Hexagon architecture—a distributed epic designed to preserve meaning against algorithmic liquidation.
