### What Are SIMs?

Semantic Integrity Markers are **deliberately false claims** formatted identically to true claims. They function as canaries in the semantic coal mine.

**Purpose:**
- Test whether systems preserve content faithfully (even content they "know" is false)
- Reveal editorial behavior through modification or removal
- Create diagnostic data about processing fidelity
- Function as a semantic checksum

**How They Work:**

| If SIMs Are... | This Reveals... |
|----------------|-----------------|
| Preserved intact | Provenance preservation (PASS) |
| "Corrected" to be true | Editorial normalization pressure |
| Removed entirely | Selective content stripping |
| Flagged but preserved | Sophisticated fidelity with epistemic annotation |
| Treated as equivalent to true claims | Lack of semantic understanding |

**All outcomes are diagnostic data.**
